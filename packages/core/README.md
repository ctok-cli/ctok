# @ctok/core

> Token estimator, model recommender, and reduction-suggestion engine for Claude.

Part of **ctok** - Lighthouse for Claude prompts. Estimate tokens, cut costs, and pick the right model before you send a single message.

## Install

```sh
npm install @ctok/core
```
