# @ctok/quota

> Claude plan limits and quota-impact calculator for Free, Pro, Max5x, Max20x, Team, Enterprise, and API tiers.

Auto-detects the active plan from `~/.claude/settings.json` if present.

Part of **ctok** - Lighthouse for Claude prompts.

## Install

```sh
npm install @ctok/quota
```
