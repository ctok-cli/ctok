# @ctok/refiner

> 7-pass heuristic prompt refiner plus a 0-100 specificity score. Fully offline.

Passes: filler strip, vague-verb replace, structure scaffold, dedup, file-ref compression, output-format hint, negative-collapse.

Part of **ctok** - Lighthouse for Claude prompts.

## Install

```sh
npm install @ctok/refiner
```
