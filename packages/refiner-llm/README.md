# @ctok/refiner-llm

> Optional Claude-powered semantic prompt refiner via the Vercel AI SDK.

Use only when the heuristic `@ctok/refiner` is not enough. Requires `ANTHROPIC_API_KEY`.

Part of **ctok** - Lighthouse for Claude prompts.

## Install

```sh
npm install @ctok/refiner-llm
```
