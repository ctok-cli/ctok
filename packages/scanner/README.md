# @ctok/scanner

> Project-directory scanner with per-ecosystem excludes (Node, Flutter, iOS, Android, Rust, Go, Python, Ruby, PHP, JVM, Swift, .NET, Elixir).

Part of **ctok** - Lighthouse for Claude prompts.

## Install

```sh
npm install @ctok/scanner
```
